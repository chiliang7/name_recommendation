# 寶寶取名助手 👶

台灣繁體中文取名 web app:三才五格 × 生肖字根 × 康熙筆畫 × 英文名推薦。

## 線上版(GitHub Pages)

**https://chiliang7.github.io/name_recommendation/**

無需伺服器——透過 [Pyodide](https://pyodide.org)(WebAssembly 版 Python)直接在瀏覽器內執行同一套 Python 計算引擎與 SQLite 字庫。首次開啟需下載引擎(約 10-20MB,之後有快取)。部署由 GitHub Actions 自動完成:push 到 main → CI 下載 Unihan、重建字庫(含筆畫驗證,失敗即擋下)、組裝靜態站、發佈 Pages。

## 本機快速開始

```bash
cd naming-app
python3 server.py        # 純標準庫,免安裝任何套件
# → http://localhost:8000
```

或使用 FastAPI 版(功能相同,多了 /docs 自動 API 文件):

```bash
pip3 install fastapi uvicorn
uvicorn app.main:app --reload --port 8000
```

## 功能

1. **三才五格**:輸入姓名 → 天/人/地/外/總格數字、五行、81 數理吉凶、三才(依五行生剋推導,例:土金火 → 凶)。
2. **生肖字根驗證**:依出生年生肖(2026 = 馬)檢查名字每個字的部件——三合/六合字根加分(馬 → 虎、狗、羊),對沖/相害/相刑與傳統忌用字根示警(馬 → 忌鼠、田、雙口…)。
3. **生肖選字推薦**:從 472 個台灣常用命名字中,剔除含忌用字根者、依生肖契合度排序。
4. **英文名推薦**:依名字拼音與字義推薦(俊/駿 → Jun、Justin;恩 → Grace)。
5. **名字推薦(平仄音韻)**:輸入姓氏+單/雙名+性別,從生肖合格字組合出完整名字,依音韻原則打分排序——三字同調扣分、有平有仄即加分(取其精神、避免單調即可,不苛求嚴格相間)、尾字平聲/開口音(ㄚ/ㄤ/ㄥ)加分、疊聲母/疊韻/同音相連扣分、雙三聲提示變調;同時排除三才大凶與人/地/總格數理凶的組合。若某姓氏因五格結構單名必為三才凶(如郭姓 15 畫),會顯示警示並建議改用雙名。結果採**分數加權隨機抽樣**(高分候選池中抽 40 個,同字最多出現 3 次、同平仄型態最多佔一半),每次都不同,可按「🎲 換一批」重抽。可調參數:**用字冷門度**(常見優先/均衡/偏冷門,依香港小學字級+Big5 分級估計常用度)、**吉度門檻**(嚴選=三才大吉且人地總數理皆吉/吉以上/寬鬆,無符合時自動放寬並提示)、**全名總筆畫上限**(現代筆畫、實際書寫)、**想用的字**(如「程,睿」,最多 4 個,逗號/頓號/空白/注音等分隔皆可)——推薦的名字必含指定字,多個指定字時名額平均分配;指定字不在精選池也可以(直接查字庫),犯生肖忌用仍會列出但附警語;若指定字與該姓氏筆畫結構在目前吉度下無合格組合,會提示改用寬鬆門檻、**不想用的字**(黑名單,最多 20 個,一律排除;與想用的字衝突時報錯)。點擊結果可跳轉完整分析。諧音仍請自行唸唸看。

## 筆畫標準

- 預設**康熙筆畫**(台灣姓名學公版):由 Unihan `kRSUnicode` 部首+餘筆推導,天然符合部首歸正(氵→水4、艹→艸6、阝→阜8/邑7、月→肉6…),數字一~十以字義計畫,另有 `data/overrides.json` 覆寫表(王=4 等)。
- **現代筆畫**採 CNS11643 全字庫官方值(`data/cns/`),與教育部國語辭典一致(陳11、福14——注意台灣官方慣例礻歸示 5 劃、衤歸衣 6 劃),查無時退回 Unihan kTotalStrokes;名字推薦的「總筆畫上限」用的是這個(含姓氏)。
- 內建 29 個常用字驗證(陳16、駿17、王4…),建庫時自動檢查。

## 資料庫

字庫涵蓋 Big5 全部 13,064 個繁體字(常用+次常用),含部件拆解(IDS 遞迴展開,支援「雙口」這類計數規則)。

重建資料庫(需 `data/unihan/`(解壓自 Unihan.zip)與 `data/ids.txt`):

```bash
python3 data/build_db.py
```

## 專案結構

```
app/wuge.py            三才五格引擎(五格、五行、81數理、三才生剋評分)
app/zodiac.py          生肖規則(三合六合沖害刑自動推導 + 傳統喜忌)
app/core.py            共用邏輯(分析、選字、英文名)
app/main.py            FastAPI 後端
server.py              純標準庫後端(免安裝)
static/index.html      前端(單頁)
data/build_db.py       建庫腳本(Unihan + IDS → SQLite)
data/shuli_81.json     81 數理吉凶表
data/zodiac_extra.json 12 生肖傳統喜忌字根
data/name_pool.json    精選命名字庫(472 字,含性別與寓意)
data/english_names.json 拼音/字義 → 英文名對照
data/overrides.json    康熙筆畫覆寫(數字、王=4…)
db/naming.sqlite3      字庫(建庫產物)
```
